Q: What does the name 'SOFLM' stand for?

A: Simple Online Football League Manager.

Q: What is that all about then?

A: I wanted to create an online soccer management game which could be played in short burts in a back to basics manner.

Q: Do I need to be online at a certain time to play the game then?

A: No, the game is not played live as such, you issue a set of 'what if' commands as part of your team selection, and the game will carry these out whether you are logged in or not, please read the rules for further information.

Q: So the game does not have a football manager style time commitment then?

A: No, you can log in for a few minutes at a time to set up your match day squad for the upcoming game, then forget about it if you want, the game is designed to be simple but challenging.
